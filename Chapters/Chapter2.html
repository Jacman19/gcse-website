Chapter 2: Iteration

Iteration is another name for repetition. In computer programs iteration is implemented with loops. There are many different types of loops such as: FOR Loops, WHILE Loops and DO UNTIL Loops. Iteration is very important in coding as it can be used to repeatedly check for a condition to be true or tocycle commands.
